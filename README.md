# Roblox Financial Analysis (Q1 2024 – Q1 2026)
Analysis of Roblox's quarterly SEC 10-Q filings across three periods using Python, SQL, and Tableau.

# Tools
Python/pandas - data cleaning and normalization
SQL - joins and aggregations
Tableau - visualizations

# Data Source
SEC EDGAR 10-Q filings for Roblox Corporation (Q1 2024, Q1 2025, Q1 2026)

# Key Questions
How has Roblox's revenue grown year over year?
How have operating expenses changed relative to revenue?
Is Roblox moving toward profitability?


# Structure
Dataset/ - raw Excel files from EDGAR
clean.py - data cleaning and normalization
queries/ - SQL analysis
dashboard/ - Tableau workbook
